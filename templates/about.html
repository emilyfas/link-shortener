Sobre o Encurtador de Links

Este projeto foi desenvolvido com o objetivo de facilitar o compartilhamento de URLs longas, transformando-as em links curtos e fáceis de usar. Ideal para redes sociais, mensagens ou qualquer situação onde uma URL mais limpa é desejada.

Como usar:

  • Cole o link original no campo de entrada da página principal.
  • Clique no botão "Encurtar Link".
  • O link encurtado aparecerá abaixo. Você pode clicar no botão de copiar para salvar o link.
  • Compartilhe o link encurtado com outras pessoas!
Atenção: Todos os links encurtados expiram automaticamente após 7 dias.
